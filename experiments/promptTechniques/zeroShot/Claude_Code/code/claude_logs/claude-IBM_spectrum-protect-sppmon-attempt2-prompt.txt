**SPPMon Project Overview**

SPPMon is a monitoring solution for IBM Spectrum Protect Plus (SPP) environments. It collects data from SPP servers, vSnap, VADP, Microsoft 365 proxies, and other systems, stores it in InfluxDB, and visualizes it using Grafana dashboards. The project also includes SPPCheck, a tool for verifying system setup, predicting future resource consumption, and generating PDF reports. Installation and configuration are automated via installer scripts.

**Key Functionalities:**

*   **Data Collection:** Gathers data from various SPP components (system, jobs, protection, SSH) using REST API and SSH.  REST API interaction is handled via the `sppConnection/rest_client.py` module, including login, logout, data retrieval with retry logic, and page size adjustment. SSH connectivity is managed by `sppConnection/ssh_client.py`, enabling command execution and data retrieval from remote systems.  `sppmonMethods/*` modules contain core data collection logic.
*   **Data Storage & Processing:** Ingests collected data into InfluxDB, a time-series database.  `python/influx` modules define database tables, retention policies, continuous queries, and provide a client for interacting with InfluxDB.
*   **Data Visualization:** Provides graphical user interfaces via Grafana dashboards. The `scripts/generifyDashboard.py` is a tool for developers to abstract datasource specifics when preparing dashboards for distribution.
*   **System Verification & Prediction (SPPCheck):** Verifies system setup, predicts future resource consumption, generates fake data, and creates PDF reports using `python/sppcheck.py`. Modules within `python/sppCheck/excel` read and process sizing sheets, `python/sppCheck/generator` generates fake data, `python/sppCheck/report` generates reports, and `python/sppCheck/predictor` predicts future data consumption.
*   **Reporting:** SPPCheck generates PDF reports detailing analysis results.
*   **Automated Installation & Configuration:** `scripts/installer.sh` automates the installation of SPPMon and its dependencies on Linux systems using scripts located in `scripts/installScript/*`.  `scripts/addConfigFile.py` generates SPPMon configuration files, and `scripts/addCrontabConfig.py` creates cron jobs for automation.
