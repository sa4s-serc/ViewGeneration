*   **Agent Modeling:** Defines agent types (e.g., rate traders, price traders, leveragers, safe owners) with varying strategies and risk profiles to interact with the simulated NOI system.
*   **Frontend (React):** Provides a user interface for interacting with deployed smart contracts, primarily focused on CDP management.
