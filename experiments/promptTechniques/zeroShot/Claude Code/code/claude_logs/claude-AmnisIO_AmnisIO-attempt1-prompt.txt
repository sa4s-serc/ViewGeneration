AmnisIO is a project designed to bring reactive programming principles to IoT development, enabling developers to build applications for embedded devices, such as the Arduino UNO, using TypeScript. *   **PlatformIO Integration:** PlatformIO is integrated for project management, building, and deploying applications to target devices. This simplifies tasks like dependency management and flashing firmware.
*   **`rivulet` Stream Library:**  A lightweight C stream library named `rivulet` (exposed via TypeScript) forms the foundation for reactive programming. It provides the necessary tools for creating and manipulating data streams.
