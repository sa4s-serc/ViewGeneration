*   **Private Link:** Enables secure, private connections to PaaS services like PostgreSQL and MinIO, minimizing exposure to the public internet.
*   **License Server (Optional):** A Windows VM to host the dSPACE Installation Manager. Includes configurations for IaaSAntimalware and MicrosoftMonitoringAgent.
*   **Log Analytics Integration (Optional):** Integrates with Azure Log Analytics for centralized logging and monitoring.
*   **Bastion Host (Optional):** Provides secure access to VMs within the virtual network.
