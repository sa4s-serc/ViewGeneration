## Movalytics: Data Warehouse Solution for Movie Analytics - Repository Analysis Report

This report analyzes the Movalytics GitHub repository, a data warehouse solution designed for movie analytics. The project utilizes Apache Airflow for orchestration, Apache Spark for data processing, and Amazon Redshift as the data warehouse. *   **Data Loading:** Transformed data is loaded into staging tables in Redshift, and subsequently upserted into final dimension tables using SQL scripts.
*   **Orchestration:** Apache Airflow orchestrates the entire ETL pipeline, defining task dependencies and scheduling their execution.
*   **Data Quality Checks:** Data quality checks are implemented within the pipeline to ensure data integrity. A custom `DataQualityOperator` is employed for this purpose.
*   **Containerization:** Docker is used to package the application and its dependencies, facilitating consistent deployments.
*   **Infrastructure as Code:** The `docker-compose-LocalExecutor.yml` file defines the necessary services (Postgres, Airflow webserver) to run the application, enabling easy deployment and management of the infrastructure.
